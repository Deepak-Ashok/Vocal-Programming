Vocal Programming Tool

Welcome to Vocal Programming Tool. Vocal Programming Tool is your voice code typist. All you have to do is speak out the code, and the app will write it for you!

Vocal Programming Tool has certain syntax rules, and only responds to specific voice commands. Make sure you take a look at the readme document before you get started.

About

Vocal Programming Tool is a especially for visually impaired and people with upper body impairment to facilitate to learn programming concepts.

Our aim is to encourage people in coding, by teaching programming concepts and empower them to achieve the best future by giving programming theories simply with coding examples
